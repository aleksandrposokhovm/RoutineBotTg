# API Routes

Здесь хранятся Express роуты для обработки запросов от frontend приложения (Telegram WebApp).
Каждый файл отвечает за свою часть данных:
- `auth.ts`, `googleAuth.ts`: Авторизация
- `diet.ts`: Работа с питанием
- `plans.ts`: Работа с планами
- `schedule.ts`: Работа с расписанием
- `profile.ts`: Работа с профилем
