# Services

Здесь хранится бизнес-логика и интеграция со сторонними сервисами:
- `gemini.ts`: Взаимодействие с API Google Gemini (AI-ассистент, промпты, история сообщений).
- `googleCalendar.ts`: Интеграция с Google Календарем.
- `scheduler.ts`: Планировщик задач (cron).
- `timezone.ts`: Утилиты для работы с часовыми поясами.
